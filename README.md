# 货道控制板 30×30 — 固件项目

> STM32F103C8T6 货道控制板固件，兼容 M109E 售货机控制卡协议

## 项目概述

30×30 货道控制板，驱动 900 通道出货机构，通过 RS-485 通讯与上位机对接。

| 项目 | 规格 |
|------|------|
| MCU | STM32F103C8T6 (72MHz, 64KB Flash, 20KB RAM) |
| 矩阵 | 30行×30列，74HC595×8片级联驱动 |
| 通讯 | RS-485, 9600bps, 兼容 M109E 协议 |
| 电流检测 | OPA2365 + 20mΩ 采样电阻, 增益21倍 |
| 输入 | 12路光耦（10路到位检测 + 2路光幕） |
| 输出 | 4路 NMOS 开关量 |
| 供电 | DC 24V → LM2596(5V) → AMS1117(3.3V) |

## 通讯协议

兼容 M109E 售货机控制卡，20 字节定长帧，CRC16-MODBUS 校验。

| 指令 | 功能 | 说明 |
|------|------|------|
| 0x01 | 获取序列号 | 返回 12 字节板卡 ID |
| 0x03 | 查询电机状态 | 运行状态/结果/电流/时间 |
| 0x05 | 启动电机 | 通道号+类型+光幕模式+超时 |
| 0x07 | 读温度 | NTC 温度值 |
| 0x08 | 开关量输出 | 4 路 NMOS 控制 |
| 0x09 | 读开关量输入 | 4 路 DI 状态 |
| 0x0B | 开停光幕 | 光幕开关 |
| 0x0C | 读光幕状态 | 遮挡检测 |
| 0x0D | 读光幕计时 | 遮挡毫秒数 |
| 0x10 | 读温湿度 | 温度 + 湿度 |
| 0x2A | 读单通道到位 | 到位开关状态 |
| 0x2B | 读一行到位 | 一行 30 通道状态 |
| 0xFF | 设置地址 | 广播设置从机地址(1-8) |
